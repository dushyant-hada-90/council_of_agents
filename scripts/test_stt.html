Auth (GOOGLE_APPLICATION_CREDENTIALS)

Accepts the same JSON as GOOGLE_APPLICATION_CREDENTIALS on the server: service_account (JWT) or authorized_user (refresh token). For authorized_user, include quota_project_id — the client library sends it as x-goog-user-project automatically; this harness does the same.
